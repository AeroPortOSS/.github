# Security Policy

## Supported Versions

The following versions of AeroPort currently receive active security updates and patches:

| Version | Supported          |
| ------- | ------------------ |
| Main    | Yes (Development)  |

## Reporting a Vulnerability

Security is a fundamental priority for paravirtualized system drivers. If you discover a vulnerability, memory leak, or potential hypervisor containment escape, please do not open a public issue.

Instead, please report security vulnerabilities by creating a confidential disclosure directly through the GitHub Security Advisory dashboard. 

We aim to review all submissions within 48 hours and coordinate a patch cycle prior to public disclosure.
