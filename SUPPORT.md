# AeroPort Support

Welcome to the AeroPortOSS support guide. Please utilize the following pathways to resolve configuration questions or platform installation inquiries.

## Technical Issue Tracking

If you have verified a reproducible bug or an architectural regression within the code stack:
1. Navigate to the specific affected repository tracker.
2. Select "New Issue" and complete the mandatory Bug Report form template.
3. Provide full platform logs and structural hardware information.

## General Configuration and Discussion

For installation assistance, hardware compatibility questions, or community coordination:
* Please utilize the GitHub Discussions tab located within our primary `aeroport-macos` user-facing repository.
* Do not open standard tracking issues for general setup troubleshooting.
